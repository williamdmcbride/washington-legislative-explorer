About WA Legislative Explorer

Data Sources

  • Live API Data (Real-time from WA State Legislature):
  • ✅ Bills - Washington State SOAP API (LegislationService)
  • ✅ RCW Citations - Washington State SOAP API (RcwCiteAffectedService)
  • ✅ Committees - Washington State SOAP API (CommitteeService)
  • Web Research (Perplexity AI):
  • 🔍 WAC (Washington Administrative Code) - No official API exists
  • 🔍 County Codes - No official APIs exist
  • 🔍 Research & News - Real-time web search
  • RAG-Powered Search (Building Codes):
  • 🏗️ Building Codes - Retrieval-Augmented Generation using local vector database
  • Searches 796 sections across 12 Washington Administrative Codes (WAC Title 51)
  • Includes: Fire, Building, Residential, Mechanical, Plumbing, Energy, and Administrative codes
  • Provides answers with specific code citations and page numbers
  • Combines semantic search with Claude AI

Washington State Legislature provides several SOAP-based web services for accessing legislative information. Below is a complete list of available APIs and how this tool uses them.

LegislationService ✅ In Use

Provides information about bills, amendments, sponsors, and bill status. Includes search capabilities and detailed bill information.

RcwCiteAffectedService ✅ In Use

Returns legislation that affects specific RCW (Revised Code of Washington) citations. Shows which bills modify or reference particular laws.

CommitteeService ✅ In Use

Provides committee information including names, members, meeting schedules, and committee assignments.

SessionLawService ○ Not Used

Provides access to session laws - bills that have been enacted into law during a legislative session.

SponsorService ○ Not Used

Detailed information about bill sponsors, including legislator profiles and sponsorship history.

LegislativeDocumentService ○ Not Used

Access to legislative documents including bill drafts, amendments, fiscal notes, and other official documents.

CommitteeActionService ○ Not Used

Tracks committee actions on bills including hearings, votes, and recommendations.

Note: There are NO official APIs available for RCW text, WAC (Washington Administrative Code) text, or County Codes. For these, we use Perplexity AI to search the web and cite official sources.

Building Codes: Uses RAG (Retrieval-Augmented Generation) with a local vector database created from 12 official WAC Title 51 PDFs (Fire, Building, Residential, Mechanical, Plumbing, Energy, and Administrative codes). This is not an API but provides fast, accurate searches across all building codes with specific code and page citations.

How It Works

  • Live API tabs connect directly to official WA State Legislature databases.
  • Web Research tabs use Perplexity to search the internet and cite sources.
  • All web research results include clickable source links.

  • Building Codes RAG: The Building Codes tab uses Retrieval-Augmented Generation (RAG) technology. All 12 WAC Title 51 building codes have been processed into a searchable vector database:

    1. WAC 51-04 (Policies), 51-05 (Surcharges), 51-06 (Records), 51-16 (Guidelines)
    2. WAC 51-50 (Building Code), 51-51 (Residential Code), 51-52 (Mechanical Code)
    3. WAC 51-54A (Fire Code), 51-55 (Wildland Code), 51-56 (Plumbing Code)
    4. WAC 51-11C (Energy Commercial), 51-11R (Energy Residential)

    When you ask a question, the system searches all codes simultaneously and retrieves the most relevant sections, then Claude generates an answer citing exact codes and page numbers.

What This Means

  • Bills, RCW, and Committees data is official and real-time.
  • WAC and County Code information comes from web searches with source citations.
  • Always verify critical information with official sources.

Limitations & Disclaimers

  • Live API data is official but may have delays during updates.
  • Web research is accurate but should be verified for legal/compliance decisions.
  • For official WAC codes, visit: apps.leg.wa.gov/wac
  • For official county codes, visit your county's website.
  • This tool is for research purposes, not legal advice.
  • Building Codes RAG Limitations:
  • Based on 2021 editions of WAC Title 51 codes (may not reflect newer amendments)
  • Searches semantic meaning, not exact keyword matching
  • Always verify critical requirements with official sources
  • For official codes: apps.leg.wa.gov/wac/default.aspx?cite=51
